# Tipo-Abstrato-de-Dados-Pilha
Este repositório contém um programa modular em C que implementa e testa uma TAD Pilha, utilizando a política LIFO (Last In First Out) com uma lista ligada com nodo cabeça.

O programa modular que cria e testa a TAD PIlha é composto pelos seguintes arquivos:

- pilha.h: protótipos das funções, que criam, modificam e destroem a pilha. Este arquivo foi desenvolvido pelos professores do departamento de informática da Universidade Federal do Paraná, sendo eles Andre Ricardo Abed Gregio, Marcos Alexandre Castilho e Luis Carlos Erpen de Bona.
- pilha.c: implementação de todas as funções existentes em pilha.h, feito por Jean Pablo Lopes Teixeira.
- testa_pilha.c: desenvolvido por Marcos Alexandre Castilho, testa as implementações de pilha.c verificando se a pilha é criada corretamente, se os dados são obtidos sem problemas de ponteiros, se a pilha manipula corretamente seus dados no topo (nodo cabeça) e se a pilha entende que está vazia ou não.

Por fim, este programa tem o propósito apenas de testar a criação da Pilha, mas pode ser adaptado para utilizar a Pilha em outros contextos que envolvam utilização de lista ligada com nodo cabeça.
